Using Animancer

This user manual describes all the features of Animancer and how to use them.

To use this plugin you will first need to download it from the Unity Asset Store:

  1. Back up your project first and if you are updating from an older version of Animancer it is recommended that you fully delete the old version before importing the new one. This is a good general practice when updating or installing any asset.
  2. If updating from an older version of Animancer you will also need to restart Unity after importing due to this bug.
  3. The navigation bar at the top of this page has links to Try Animancer Lite or Buy Animancer Pro. Feel free to start with the Lite version to try it out and only upgrade to Pro if you actually need the features. The upgrade process is designed to be totally seamless so it doesn't interrupt your work any more than necessary.
  4. The Asset Store page will lead you to open it within the Unity Editor so you can download the package.
  5. Once the download is complete you can import Animancer into your project and begin using it.

The Assets/Plugins/Animancer/Demos folder contains some examples that show Animancer in action.

Playing Animations To play animations using a script simply add an AnimancerController component and call it's Play(AnimationClip) method.
Animancer Controller Types There are several different kinds of AnimancerController and you can make your own to add or modify functionality.
The Animancer Inspector AnimancerController components are primarily controlled by scripts, however in Play Mode in the Unity Editor they show details about their current states so you can observe what they are doing and fiddle with them manually for debugging purposes.
Smooth Transitions You can smoothly transition between animations using CrossFade and create AnimancerTransition assets to define transition parameters in the Unity Editor instead of hard coding everything.
Animancer States The most common type of AnimancerState is a ClipState which plays a single AnimationClip and is registered in an internal dictionary using that clip as the key.
Animator Controller States You can play and blend between multiple AnimatorController assets using Animancer.
Animation Mixers How to use AnimationMixers to interpolate between multiple animations based on a parameter.
Animation Layers Use layers to define how groups of animations blend with each other.
