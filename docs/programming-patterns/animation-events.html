Animation Events

You will often have the need to do things at a certain time during an animation such as playing a footstep sound each time a foot touches the ground during a walk cycle or applying the effects of an attack at a specific point during a sword swing. You could script these things using coroutines or other timers, but using Animation Events allows you to more easily coordinate the script with the animation. The Unity Manual explains how to set up Animation Events using the Animation Tab of a model's import settings or the Animation Window.

Using Unity's Animation Event system is the same with Animancer as without it; you simply attach a script to the same object as the Animator component and the engine will look for methods with appropriate names in that script. It can be convenient to have your event listener script inherit from AnimancerController for organisational purposes, but such events don't actually have any direct interaction with Animancer.

Simple Events

SimpleEventAnimancerController is a component with an OnEvent delegate which is triggered by Animation Events called Event and cleared whenever Play or CrossFade is called. This means you can start an animation then register a callback for when its event occurs, and it will automatically assert that you aren't trying to set multiple callbacks and that the playing clip actually has an event with that name (while in the Unity Editor). It inherits from EventfulAnimancerController and mainly serves as an example of how you can implement your own events, but it could be useful in a simple game if you set up all of your animations with only one event each that uses Event as the Function name (Attack script uses the event to indicate the hit time, Walk script uses it to play footstep sounds, etc).

The EventfulAnimancerController script inherits from the base AnimancerController, meaning that it does not have the Animations array and Play Automatically toggle from LegacyAnimancerController. If you would like to give it those fields, you can simply change its inheritance in the script (Assets/Plugins/Animancer/EventfulAnimancerController.cs).
