Frequently Asked Questions

Question Answer
What's the difference between Pro and Lite? Animancer Pro includes more features which the Lite version only includes in the Unity Editor for trial purposes. See the Feature Comparison for the actual list of differences.
Does it come with the source code? Animancer Pro includes the full source code. The DLLs are empty dummies to overwrite the ones from the Lite version. See Directory Structure for more information.
Why is Unity giving an error when I select an object with an AnimancerController? If the error is "Instance of AnimancerControllerEditor couldn't be created. The script class needs to derive from ScriptableObject and be placed in the Assets/Editor folder." then simply restarting Unity will fix it. More information can be found here.
Does Animancer support Inverse Kinematics? Technically yes, but see here for some important details.
Why am I getting an ArgumentException when using a custom state type? You probably didn't set the _BasePlayable as described here.
What is the performance like? The performance results are close enough not to matter. General frame rate is slightly faster in Animancer while instantiation is slower. More details are available in the Performance Benchmarks section.
