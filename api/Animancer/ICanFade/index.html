Properties

Name Value Summary
FadeSpeed float
The speed at which this node is fading towards the Animancer.ICanFade.TargetWeight.
Parent IAnimationMixer
The Animancer.IAnimationMixer which this node is connected to.
TargetWeight float
The blend Animancer.ICanFade.Weight which this node is fading towards.
Weight float
The current blend weight of this node. Setting this property will also set the Animancer.ICanFade.TargetWeight which cancels any fade currently in progress. If you don't wish to do that, use the Animancer.ICanFade.SetWeight(System.Single) method instead.
