AnimancerLayer Class

Summary

A layer on which animations can play with their states managed independantly of other layers while blending the output with those layers.

This class can be used as a custom yield instruction to wait until all animations finish playing.

CrossFade(AnimancerState, float) AnimancerState
[Pro-Only] Starts fading all other animations to weight 0, starts fading the 'state' to weight 1, and returns it.

Note that if the 'state' is already playing and fading in with less time remaining than the 'fadeDuration', this method will do nothing and allow it to complete the existing fade.
Dispose() void
