public interface IPlayableWrapper

Properties

Name Value Summary
KeepChildrenConnected bool
Indicates whether child playables should stay connected to the graph at all times.

If false, playables will be disconnected from the graph while they are at 0 weight to stop it from evaluating them every frame which is generally more efficient.
Parent IPlayableWrapper
The object which receives the output of the Animancer.IPlayableWrapper.Playable.
Playable Playable
The UnityEngine.Playables.Playable managed by this object.
Speed float
How fast the UnityEngine.Time is advancing every frame.

1 is the normal speed.

A negative value will play the animation backwards.

Animancer Lite does not allow this value to be changed in a runtime build.
